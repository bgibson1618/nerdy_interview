The question driving all 22 experiments

Self-organization, three ways / the throughline

Three ways roster agents coordinate — with no orchestrator.

PatternCoordinationCommunicationResult

Pattern 1 · work-division / atomic claim

Exactly-once under stress — work-stealing for free.

Workers drain a shared board; the only coordination is an atomic flock claim (read-find-mark-write). Two workers never take the same task.

0 double-claims · 10 workers × 100 tasks
10/9/7/4 speed-skewed workers split with no balancing logic — emergent load-balance
Pattern 2 · leader-election / replication earned this

Term-based, highest-id-wins over A2A broadcast. 28/30.

Replication earned two findings

  • Liveness bug: a non-max candidate that missed the single COORDINATOR broadcast stranded forever (no retry). Fixed.
  • Transport ceiling: the node loop spins as fast as messages arrive — an ungated heartbeat storms send … all, each a forked bash+jq+flock+inotify.

After fixes

N=3 perfect N=5 perfect · incl. failover N=7 steady perfect N=7 + failover · 3/5 soft

Only the failover case stays soft — bursty jitter the timing budget can't always cover.

Pattern 3 · shared-blackboard / stigmergy

Coordination through shared state — no messages at all.

  • Agents contribute to a shared multi-level store and react to its state — 0 messages.
  • A completeness-gated synthesis slot, claimed atomically, guarantees the final reduction happens exactly once.
  • No ceiling — it's low-frequency.
20/20 zero violations

no lost work · no double-work
synthesis-once · correct vs ground truth
synthesis > completeness · emergent balance

The cross-cutting insight / what it all reduces to

The atomic claim is the universal primitive.

  • Work-division, the blackboard, and the A2A inbox all reduce to one race-free flock op. Get it right and leaderless coordination falls out.
  • Reliable broadcast enables consensus — the file-backed inbox provides it; election converges.

The ceiling is the TRANSPORT, not the protocol

High-frequency message-passing (heartbeats) hits the fork-per-message cost; low-frequency stigmergy doesn't. Shared-state scales where chatty message-passing does not.

To lift it, change the transport (persistent/batched), not the algorithm.

Unanimous — all three agreed leader = claude (id 30); 10 typed envelopes. The leader then assigned a task to a follower over A2A.

Zero-message stigmergy

Three agents built a 6-section fact-sheet through a shared board — 0 A2A messages, emergent division, synthesis-once.

Faster backends won the section races; honest work-stealing — codex got shut out.

Takeaways / what to build first

What an implementer should take away.

01Route by measured strength, not reputation — claude for framing, codex for bounded review/drift, gemini for research/judging — never the security gate.
02Consensus / diversity as insurance, not default — it pays only on tasks with divergent blind spots.
03Ground truth + replication earns the surprising findings — the codex-for-drift reversal, the no-herding result, the transport ceiling — none were guessable up front.
04Lift the transport before scaling chatty protocols. Leaderless coordination works today at small N; the atomic claim is the thing to make first-class.
