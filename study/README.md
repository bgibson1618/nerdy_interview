# Study material

Code-review interview prep (Nerdy / Varsity Tutors). The main plan lives at the repo root
(`../NERDY_STUDY_PLAN.md`); this folder holds the standalone practice material it references.

- **`drills/`** — self-contained code-review drills: planted-bug exercises (a buggy file +
  an answer key) and PR-candidate code (buggy branches to review in the GitHub UI).
- **`exercises/`** — targeted skill exercises, e.g. JS/TS gotcha walk-throughs.

Each drill is standalone: read the prompt, do the review out loud / on paper, then check the key.
